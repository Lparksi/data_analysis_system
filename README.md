## 简介
本项目用于自动化计算各类考试成绩的三率一分，支持选取前 n 百分比，自定义各科总分。  
配置文件：**attachment/template1.ini**   
模板文件：**attachment/template1.xlsx**

## 使用
### 安装依赖
```bash
pip3 install -r requirements.txt
```
### 选择使用方式
#### WEB
```bash
python3 app.py
```
#### 单文件
```bash
python3 single_Py.py
```
---
## 上传原文件和配置文件
将你的成绩文件修改为和模板文件一样的格式，然后上传到网页上，选择配置文件，点击提交即可。  
配置文件的修改见配置文件内的注释。
